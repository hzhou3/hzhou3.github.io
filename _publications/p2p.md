## Description

A BitTorrent-like peer-to-peer based file sharing prototype system is built in this project.

## What learned

1. Design different functionalities for peers, each peer has his/her own client thread and server thread to recieve and send file chunks...
2. Design communication protocols.
3. Any peer is able to upload and and download at the same time.
4. Leaves of peers don't affect downloading as long as one of peer holds the demanded chunk.
5. Unfinished downloading can be resumed.
