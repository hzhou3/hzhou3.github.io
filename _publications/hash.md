## Description
Design a concurrent hash table on GPU using CUDA. In this project, a chaining design is used to solve conflicts.

## What Done

1. Study different ways to solve conflicts.
2. Utilize atomic operations for lock-free updates (insertion and deletion).
3. Understand warp divergence and memory access pattern.
