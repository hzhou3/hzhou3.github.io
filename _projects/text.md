## Description

This is a project for Machine Learning. In this project, students are given a bunch of files each with one of nine classes. The file is organized in Bag of Word format. Given the training, validation, and testing sets, students need to perform feature selection or feature weighing, then train four different classifiers. The four classifiers are Naive Bayes, KNN, SVM, and Neural Nets. Hao's Naive Bayes classifier achieved the best accuracy in the class and made a record (90.78% accuracy - 2916 files in the testing set).

## What done

1. TF-IDF is used for feature weighing, and different feature selection methods are studied: Information Gain, Mutual Information, chi-square test...

2. Based on Bayes theorem to build a Naive Bayes classifier, and Laplace smoothing is used.
3. Support Vector Machine is studied in this project, and it turned out linear SVM is a better fit.
4. KNN is also learned, and different distance metrics are tested.
5. Neural Nets are learned as well. Linear and convolutional layers for representation learning, and other layers like dropout and batch normalization are also studied.
